破解Mozilla Firefox地址栏

在Firefox的地址栏中，您可以通过在搜索词之前或之后键入特殊字符来限制结果：

- ^ - 匹配浏览历史记录

- * - 匹配书签。

- % - 匹配当前打开的标签页。

- # - 匹配页面标题。

- @ - 匹配Web地址。
