Docker docker.com

是一个供开发者和系统管理员开发、交付和运行应用程序的平台。

Moby github.com

是由 Docker 创建的用于组装专用容器系统的开放框架。

podman podman.io

是一个无守护进程的容器引擎，用于在您的 Linux 系统上开发、管理和运行 OCI 容器。

gVisor github.com

是一个用 Go 编写的用户空间内核，在主机操作系统和应用程序之间提供安全的隔离边界。

Buildah github.com

一个有助于构建 OCI 容器镜像的工具。

runc github.com

是一个根据 OCI 规范生成和运行容器的 CLI 工具。

containerd containerd.io

是一个行业标准的容器运行时，强调简单性、健壮性和可移植性。

Kata 容器 github.com

是一个致力于构建轻量级虚拟机标准实现的开源项目和社区。

dive github.com

一个用于探索 docker 镜像、层内容以及发现缩小 Docker 镜像大小的方法的工具。

Kubernetes kubernetes.io

是一个用于自动化部署、扩展和管理容器化应用程序的开源系统。

K3s k3s.io

是一个高可用、经认证的 Kubernetes 发行版，专为无人值守、资源受限、远程位置或物联网设备内的生产工作负载而设计。

k9s github.com

提供一个终端 UI 来与您的 Kubernetes 集群进行交互。

kops kops.sigs.k8s.io

是让生产级 Kubernetes 集群启动并运行的最简单方法。

kpack github.com

一种 Kubernetes 原生构建服务，可自动从源代码创建和更新容器镜像。

OpenShift open-shift.com

是一个专注于开发者体验和应用程序安全的 Kubernetes 发行版。

Rancher rancher.com

是一个用于管理容器的开源平台。

kaniko github.com

一个用于从 Dockerfile 在容器或 Kubernetes 集群内部构建容器镜像的工具。

BuildKit github.com

是一个并发、缓存高效且与 Dockerfile 无关的构建工具包。

CRI-O cri-o.io

是 Kubernetes 容器运行时接口的基于 OCI 的实现。

Harbor goharbor.io

是一个开源的可信云原生注册表项目，用于存储、签名和扫描内容。

Helm github.com

一个简化安装和管理 Kubernetes 应用程序的工具。

Skaffold github.com

一个命令行工具，可促进 Kubernetes 应用程序的持续开发。

Argo CD github.com

是一个用于 Kubernetes 的声明式、GitOps 持续交付工具。

Traefik traefik.io

一个现代化的 HTTP 反向代理和负载均衡器，使部署微服务变得容易。

Istio istio.io

一个连接、管理和保护微服务的开放平台。

Envoy envoyproxy.io

是一个专为云原生应用程序设计的开源边缘和服务代理。

eBPF ebpf.io

可以在操作系统内核中运行沙盒程序。

Cilium github.com

是一款开源软件，用于在应用程序工作负载（如应用程序容器或进程）之间透明地提供和保护网络连接及负载均衡。

Falco falco.org

是一种行为活动监视器，旨在检测应用程序中的异常活动。
